What's Changed?

1. Browser Support
Bootstrap websites will only be supporting IE10+ and iOS 7+
2. REM Support
Switched from px to rem as the primary CSS unit
3. Sass
Switched from Less to Sass for source CSS files. No more Custom Builder.
4. Additional Breakpoint (xl)
Bootstrap now has 5 breakpoint tiers with the ability to customize.
5. Reboot CSS
Resets, or standardizes, native html elements across all browsers without the need for vendor specific classes.
6. Removed Glyphicons
No longer are you stuck with multiple icon sets, use what you want. SVG or Font icon sets preferred.
7. Grid
Bootstrap now supports flexbox by default.
